Table with values

This table is used for situations where you need a right aligned value, or a total along the bottom. Both of these are achieved by simply adding .amount to a cell, or .total to a row.

Budget table
Label £00.00
Total £00.00 
<table class="full-width">
	<thead>
		<tr>
			<th scope="col" colspan="2">Budget table</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>Label</td>
			<td class="amount">£00.00</td>
		</tr>
		<tr class="total">
			<td>Total</td>
			<td class="amount">£00.00</td>
		</tr>
	</tbody>
</table>

Collapsible table

Tables can include collapsible rows by simply adding the correct classes and embedding a table within a cell.

th th
td td
td td
td td
td td
td td
td td 
<table class="full-width">
	<thead>
		<tr>
			<th scope="col">th</th>
			<th scope="col">th</th>
		</tr>
	</thead>
	<tbody>
		<tr class="table-collapse--toggle">
			<td>td</td>
			<td>td</td>
		</tr>
		<tr class="table-collapse">
			<!-- colspan is important to make the embedded table full width -->
			<td colspan="2">
				<!-- .full-width class is important to make the embedded table full width -->
				<table class="full-width">
					<tr>
						<td>td</td>
						<td>td</td>
					</tr>
					<tr>
						<td>td</td>
						<td>td</td>
					</tr>
				</table>
			</td>
		</tr>
		<tr class="table-collapse--toggle">
			<td>td</td>
			<td>td</td>
		</tr>
		<tr class="table-collapse">
			<!-- colspan is important to make the embedded table full width -->
			<td colspan="2">
				<!-- .full-width class is important to make the embedded table full width -->
				<table class="full-width">
					<tr>
						<td>td</td>

			</td>
		</tr>
	</tbody>
</table>
