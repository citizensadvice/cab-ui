New grid layout

The default page width is 1080px with a 12-spanumn grid. It is recommended for most content to use the two-thirds (content first) layout. This restricts line length to a more managable size and encourages a better reading pattern, improving cognition.

The layout for pages should be 12 spanumns (full width), 3 or 4 spanumn, other variations can be used if needed.
